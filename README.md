# finance-tools

เครื่องมือคลีน &amp; ตรวจสอบข้อมูลการเงิน (ใช้ในเบราว์เซอร์ ไม่มีเซิร์ฟเวอร์ — ข้อมูลไม่ออกนอกเครื่อง)

**เปิดใช้งาน:** https://nantawan-nan.github.io/finance-tools/

## โครงสร้าง
- รองรับ 2 บริษัท: M Bark / Benya (สลับด้านบน)
- 3 Stage · 5 เครื่องมือ:
  - **Stage 1 (บัญชี)** — จัดฟอร์แมต Big Seller · ตรวจ IV ครบไหม
  - **Stage 2 (การเงิน)** — Map ลูกหนี้คงค้าง · จับยอด Settlement
  - **Stage 3 (การเงิน)** — กระทบยอดถอนเงิน (ผ่านเช็ค)
- ทุกเครื่องมือ: อัปไฟล์ (.xlsx/.xls/.csv) → ประมวลผล → ดาวน์โหลด Excel/CSV หรือคัดลอก

## หมายเหตุ
ตอนนี้เป็น **โครงร่าง (shell)** — อ่าน/แสดง/ส่งออกไฟล์ได้แล้ว
logic เฉพาะของแต่ละเครื่องมือ (คลีน/แมพ/ตรวจสอบ) จะทยอยใส่ตามไฟล์ตัวอย่างจริง

แก้ทั้งหมดที่ `index.html` ไฟล์เดียว
