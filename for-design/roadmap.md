## 2026-06-25

### หน้าสรุปงานขาย
- ยอดขายมีช่องทางไหนบ้าง
- ขายอะไร
- ยอดกี่บาท
- กี่คำสั่งซื้อ
- (ยึดข้อมูลจากบิ๊กเซลเลอร์)

### สต๊อกสินค้า + จับสต๊อก
- เพิ่มรูปสินค้าเข้าไปเพื่อความสวยงาม
- ออกแบบระบบสต๊อก
  - ขอยอดตั้งต้น
  - รายงานสต๊อกทุกสิ้นเดือน:
    - ยกมา
    - ขายระหว่างเดือนกี่บาท
    - รับเข้าระหว่างเดือนกี่บาท
    - คงเหลือเท่าไหร่
  - จุดสั่งซื้อ
  - ตรวจยอดต้นทุนสินค้าจากบิ๊กกับไฟล์ที่เห็น

### หน้าภาพรวม — แบ่งพาร์ทเป็น บัญชี / การเงิน
- เพิ่มแท็บ:
  - รายงานขาย
  - สต๊อกสินค้า
  - ลูกหนี้การค้า
  - AP วิเคราะห์อายุหนี้เจ้าหนี้คงค้าง
  - วิเคราะห์ลูกหนี้คงค้าง

- วันที่ในวงเล็บ = วันที่จริงของศุกร์ในเดือนนั้นๆ

### Permissions — viewer/สิทธิต่างๆ ✅
- เพิ่ม checkbox grid ในหน้าจัดการผู้ใช้ → ติ๊กหน้าที่ดูได้
- เก็บใน `app_metadata.page_permissions` (jsonb array)
- null = ใช้ role default · array = filter เฉพาะที่ระบุ
- admin เห็นทุกหน้าเสมอ (override)

### ค่าใช้จ่ายประจำ — เลือกเข้ารอบจ่ายของบริษัทได้ ✅
- ตัวอย่าง: ค่าโทรครบกำหนด 25 ของเดือน แต่ผู้บริหารอยากให้เลือกเข้ารอบจ่ายปกติของบริษัทแทน
- คลิกปุ่ม "📅 วันจ่าย" ที่รายการ → modal 12 เดือน → กำหนดวันแต่ละเดือน
- chip ด่วน "ศ.1-5" + "วันที่ 15" ในแต่ละเดือน
- เก็บใน `recurring_expenses.month_schedule` (jsonb {"YYYY-MM": "YYYY-MM-DD"})
- Cash Flow link ไปตามวันที่ที่กรอกแทนวันครบกำหนดเดิม
